# Quantum Computing & Applied IT Portfolio

## 🔬 Projects Overview
This repository demonstrates the practical intersection of computational physics, software development, and cybersecurity protocols. It contains programmatic implementations of foundational quantum algorithms exploring the transition from theoretical quantum mechanics to applied IT infrastructure.

- **VQE H2 Molecule Simulation**
    Estimating the ground state energy of a hydrogen molecule using a hybrid quantum-classical Variational Quantum Eigensolver (VQE) with a hardware-efficient TwoLocal ansatz.

- **QAOA Resource Scheduling**
    Optimizing combinatorial workloads and task assignments by mapping classical constraints into an Ising Hamiltonian and navigating the state space using the Quantum Approximate Optimization Algorithm (QAOA).

- **QRNG Cryptography**
    Generating theoretically unpredictable random numbers via quantum superposition and state collapse for cryptographic key generation, compared against classical PRNG entropy.

- **Superdense Coding**
    Achieving quantum data compression by transmitting two classical bits of information through the exchange of a single entangled qubit.
